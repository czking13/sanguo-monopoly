# 三国大富翁

**项目名称**：Sanguo Monopoly（三国大富翁）
**平台**：Web（网页游戏）
**开发者**：虾仁
**开始日期**：2026-03-09

---

## 项目概述

一款以三国为背景的大富翁网页游戏，支持多人联机对战。

---

## 核心功能

### 1. 房间系统
- 创建房间（自定义房间名、人数上限、游戏规则）
- 生成邀请码
- 加入房间（输入邀请码）
- 房间列表（可选）
- 房主控制（开始游戏、踢人）

### 2. 游戏机制

#### 基础玩法
- 掷骰子移动
- 格子类型：地产、事件、机会、命运、起点、监狱等
- 购买地产、收取过路费
- 破产判定

#### 三国特色
- **武将系统**：选择武将，每个武将有独特技能
- **城池系统**：地产改为城池，占领城池获得收益
- **势力系统**：魏蜀吴，占领同势力城池有加成
- **事件卡**：三国历史事件（赤壁之战、三顾茅庐等）
- **道具卡**：锦囊妙计（火攻、连环计、空城计等）

### 3. 武将设计（示例）

| 武将 | 势力 | 技能 |
|------|------|------|
| 曹操 | 魏 | 挟天子：每次经过起点额外获得 10% 资金 |
| 刘备 | 蜀 | 仁德：经过对手城池，过路费减半 |
| 孙权 | 吴 | 制衡：每回合可重新掷一次骰子 |
| 诸葛亮 | 蜀 | 神算：可查看下一张事件卡 |
| 关羽 | 蜀 | 武圣：入狱后可立即出狱 |
| 周瑜 | 吴 | 火攻：使用火攻卡效果翻倍 |

### 4. 地图设计

- 格子数量：约 40 格
- 城池分布：按三国地理分布（中原、荆州、江东、蜀地等）
- 特殊格子：
  - 起点（洛阳）
  - 监狱（许昌监狱）
  - 机会（锦囊）
  - 命运（天命）
  - 税收（纳贡）

### 5. 联机功能

- WebSocket 实时通信
- 房间状态同步
- 玩家在线状态
- 断线重连

## 技术栈

### 前端
- **框架**：React / Vue 3
- **UI**：Tailwind CSS
- **动画**：Framer Motion
- **地图**：Canvas / SVG

### 后端
- **框架**：Node.js + Express / Fastify
- **实时通信**：Socket.io / WebSocket
- **数据库**：Redis（房间状态） + SQLite/PostgreSQL（用户数据）

### 部署
- **前端**：Cloudflare Pages / Vercel
- **后端**：云服务器 / Railway / Render
