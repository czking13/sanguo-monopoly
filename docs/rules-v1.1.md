## 一、核心规则补充

### 1.1 回合行动顺序（精确版）

回合内所有可触发事件的**完整执行顺序**（优先级从高到低）：

```
① 回合开始阶段
   ├─ 扣除/更新持续性负面效果（乐不思蜀：round--）
   ├─ 结盟倒计时更新（桃园结义：rounds--）
   ├─ 美人计冷却更新（target cooldown--）
   └─ 荀彧被动触发（+100金）

② 入狱状态检查
   └─ 若 IN_JAIL → 跳转出狱流程，成功才能继续 ③

③ 掷骰子（玩家主动或超时自动）

④ 移动阶段
   ├─ 途中经过起点：立即结算起点奖励（不打断移动）
   └─ 到达目标格

⑤ 格子效果结算
   ├─ 城池：购买/过路费
   ├─ 锦囊格：抽卡
   ├─ 天命格：触发效果
   ├─ 关卡格：停留判定
   └─ 特殊格：监狱/休息

⑥ 玩家主动使用锦囊卡（可选，限 1 张）

⑦ 玩家主动升级城池（可选，可多次）

⑧ 回合结束阶段
   ├─ 孙权被动触发（换牌选择）
   ├─ 手牌上限检查（>5 张强制弃牌）
   └─ 破产检查 → 胜负检查
```

### 1.2 骰子规则详解

**2-4人局（双骰）**：
- 掷 2 颗骰子，各自 1-6，合计 2-12
- 双数（1+1, 2+2, ... 6+6）：监狱中可用于出狱判定
- 赵云重掷：第一次结果取消，重掷 2 颗，取第二次总和

**5-8人局（单骰）**：
- 掷 1 颗骰子，1-6
- 无法通过掷骰出狱（必须缴费或使用技能）
- 太史慈技能：1颗骰子范围变为 2-7（效果：点数+1，最低2）

**张辽技能（点数+1）**：
- 双骰时：总点数+1（例：掷出 4+3=7，实际移动 8 格）
- 单骰时：点数+1（例：掷出 3，实际移动 4 格）
- 上限：不超过当前规则最大值（双骰 12，单骰 6）
  - 张辽双骰掷出 12 → 仍为 12（不变成 13）

**张辽与太史慈同时存在时（不同玩家）**：各自独立计算，互不影响。

### 1.3 锦囊卡使用时机规则（精确版）

锦囊卡的使用时机分为两类：

**A类：主动阶段使用（在回合第⑥步使用）**：
- 火攻、苦肉计、美人计、反间计、围魏救赵、以逸待劳、声东击西、草船借箭（诸葛亮特技）

**B类：响应型使用（在特定触发时机使用）**：

| 卡牌 | 使用时机 |
|------|----------|
| 空城计 | 被要求支付过路费时（步骤⑤结算前） |
| 借刀杀人 | 被要求支付过路费时（步骤⑤结算前） |
| 走为上计 | 移动结束后，格子效果结算前（步骤⑤前） |
| 瞒天过海 | 移动结束后，格子效果结算前（步骤⑤前） |

**同时有多张响应卡可用时**：玩家只能选择其中 1 张使用（弹出选项面板，超时跳过）。

**每回合只能使用 1 张锦囊卡**的补充说明：
- A类和B类合计共 1 次机会
- 若本回合已在B类时机使用了「空城计」，则步骤⑥不可再使用A类卡
- 反例：本回合没有触发任何B类时机，则步骤⑥可以正常使用1张A类卡

### 1.4 过路费支付的完整边界条件

| 场景 | 规则 |
|------|------|
| 踩到自己的城池 | 不支付过路费 |
| 踩到无主城池 | 不支付过路费，可选择购买 |
| 踩到已破产玩家的城池（释放中） | 视为无主城池 |
| 城主处于 IN_JAIL 状态 | 城主仍然收取过路费（入狱不影响收入） |
| 城主已破产 | 城池为无主，不收过路费 |
| 支付方资金精确等于过路费 | 支付后资金归零，不触发破产（资金0且还有城池不破产） |
| 结盟期间踩到盟友城池 | 免过路费（桃园结义效果） |

## 二、城池系统规则（精确版）

### 2.1 购买规则边界

| 场景 | 规则 |
|------|------|
| 不是自己回合 | 不能购买 |
| 未停在城池格 | 不能购买（只能在回合内落地时购买） |
| 资金恰好等于购买价 | 可以购买（资金归零，但有城池不破产） |
| 资金不足 | 不可购买，即使出售其他城池后够买也不行（当回合无法购买） |
| 使用「单刀赴会」天命卡 | 特例：可强制移动到任意城池并购买（若无主且资金足够） |
| 使用「七擒孟获」天命卡 | 特例：获得 1 座无主城池所有权，不需付款 |

### 2.2 城池升级规则边界

| 场景 | 规则 |
|------|------|
| 城池升级时机 | 仅限自己回合的步骤⑦（可在步骤⑤购买后立即升级） |
| 同一回合连续升级 | 可以，每次升级单独消耗资金，多级需多次确认 |
| 升级到满级（5级）后 | 无法再升级，按钮置灰 |
| 升级费不足 | 不可升级 |
| 城池被「大意失荆州」夺走 | 失去后立即变为无主，等级**保留**（原设计为重置，但v1.1调整：等级保留，给捡城玩家更多价值） |

> **v1.1 调整**：「大意失荆州」夺走的城池等级保留（原 v1.0 为破产时重置）。破产时城池释放才重置为 1 级。

### 2.3 城池出售规则

- 售价 = **累计投入的 50%**（购买价 + 所有升级费用之和的 50%）
- 出售立即生效，城池变为无主，等级重置为 1
- 强制出售（破产强制流程）与主动出售规则相同
- 不能出售给其他玩家（只能卖给系统）

**出售价格示例（洛阳）**：

| 情形 | 累计投入 | 售价 |
|------|----------|------|
| 购买后未升级 | 2000 | 1000 |
| 升到 2 级 | 3000 | 1500 |
| 升到 3 级 | 5000 | 2500 |
| 升到 4 级 | 8000 | 4000 |
| 升到 5 级 | 12000 | 6000 |

## 三、卡牌使用规则（精确版）

### 3.1 锦囊卡目标选择规则

| 卡牌 | 合法目标 | 不合法目标 |
|------|----------|------------|
| 火攻 | 任意其他存活玩家 | 自己、已破产玩家 |
| 苦肉计 | 任意其他存活玩家 | 自己、已破产玩家 |
| 借刀杀人 | 任意其他存活玩家 | 自己 |
| 美人计 | 任意其他存活玩家 | 自己、3回合冷却中的目标 |
| 连环计 | 2名不同的存活玩家 | 自己、已破产玩家、重复选同一人 |
| 反间计 | 手牌数 ≥ 1 的存活玩家 | 手牌为空的玩家 |
| 围魏救赵 | 任意其他存活玩家 | 自己 |
| 瞒天过海 | 地图任意格（1-56） | 无限制 |
| 走为上计 | 无目标 | - |
| 以逸待劳 | 无目标 | - |
| 声东击西 | 无目标 | - |

### 3.2 天命卡效果无效化情形

| 天命卡 | 被无效化的情形 | 处理方式 |
|--------|----------------|----------|
| 败走华容（入狱） | 玩家拥有关羽/司马懿且技能未用 | 弹窗询问是否使用技能 |
| 大意失荆州（失城） | 玩家无城池 | 改为损失 1000 金 |
| 乐不思蜀（停2回合） | 玩家拥有「刮骨疗毒」锦囊卡 | 触发后可立即使用刮骨疗毒移除（-500金） |
| 七擒孟获（获免费城池） | 地图上无无主城池 | 改为获得 500 金 |
| 单刀赴会（强制购买） | 无主城池为 0，或玩家资金不足 | 取消购买效果，仅移动 |

### 3.3 美人计冷却规则（详解）

- 对同一玩家使用美人计后，该目标进入 3 回合冷却期
- 冷却期内：不能再对该玩家使用美人计（按钮/该玩家图标置灰）
- 冷却计数：以「全局回合数」计算，非「该玩家的回合数」
- 例：第 10 回合对玩家 B 使用，第 10、11、12 回合对 B 免疫，第 13 回合起可再次使用

### 3.4 手牌弃牌规则

- 超过 5 张时的弃牌面板：展示所有 6 张卡牌，玩家点击想弃掉的那 1 张
- 超时（30秒）自动弃掉「最先获得的那张」（即手牌中获取时间最早的）
- 被「反间计」偷走后若低于上限，当回合不需弃牌

## 四、联盟规则（精确版）

### 4.1 联盟触发

- 唯一触发方式：天命卡「桃园结义」
- 触发者选择一名目标玩家
- 目标玩家**无法拒绝**（天命卡强制生效）
- 已有联盟的玩家触发「桃园结义」：旧联盟立即失效，与新目标结盟

### 4.2 联盟效果范围

- 互相经过对方城池：**完全免过路费**（不只是减免）
- 不能对盟友使用「火攻」「苦肉计」「美人计」「连环计（以盟友为任一目标）」「借刀杀人」
- 可以对盟友使用「围魏救赵」（交换位置，属中性）
- 天命卡「赤壁之战」（全体随机移动）不受联盟约束

### 4.3 联盟解除

| 解除方式 | 备注 |
|----------|------|
| 3 回合后自动解除 | 以触发时的全局回合数为起点 |
| 对盟友使用禁止卡牌 | 立即解除，卡牌效果照常生效 |
| 其中一方破产 | 立即解除 |
| 新「桃园结义」覆盖 | 旧联盟解除 |

## 五、加速模式与终局规则

### 5.1 加速模式触发条件（优先级排序）

以下任一条件满足，触发加速模式：

| 条件 | 说明 | 优先级 |
|------|------|--------|
| 7-8人局且全局回合数 ≥ 20 | 自动触发 | 高 |
| 任意人数局且全局回合数 ≥ 50 | 保底触发 | 中 |
| 僵局检测：连续 10 回合无人破产且所有玩家资金 > 10000 | 系统触发 | 低 |

> 全局回合数定义：所有玩家各行动 1 次 = 1 回合。例：4人局，每人各走一轮 = 1 全局回合。

### 5.2 加速模式效果

加速模式一旦触发，持续到游戏结束：

- 骰子点数 +2（双骰最大仍为 12，单骰最大仍为 6，但下限提高）
- 所有过路费 +50%
- 经过起点奖励 +1000 金（额外加，不替换原始奖励）

### 5.3 多重胜利条件结算

**标准胜利**：最后存活玩家获胜。

**快速模式（5-6人局可选）**：首位玩家总资产达到 30000 金时立即获胜。
- 总资产 = 当前资金 + 所有城池当前出售价值

**7-8人局**：首位玩家总资产达到 40000 金，或最后存活，均可触发结算。

**回合限制（2人局）**：30 回合后未分出胜负，按总资产高低判定：
- 资产高者获胜
- 资产相同：按城池数量判定
- 城池数量也相同：平局（均获得胜利积分 0.5）

### 5.4 僵局后续处理

```
僵局触发加速模式
    │
    ├─ 加速模式持续 5 个全局回合
    │
    └─ 5 回合后仍无人破产/达成快速胜利条件？
            │
            └─ 强制结算：当前总资产最高者获胜
                    ├─ 相同资产：城池多者胜
                    └─ 完全相同：记平局
```

## 六、监狱详细规则

### 6.1 入狱方式汇总

| 来源 | 触发条件 |
|------|----------|
| 天命卡「败走华容」 | 抽到该卡 |
| 未来扩展（v2.0 预留） | 三次掷出相同点数 |

### 6.2 监狱回合计数规则

进入监狱的回合**不计入**监狱等待次数（入狱当回合直接结束）。

| 入狱后的回合 | 状态 | 可选操作 |
|-------------|------|----------|
| 第 1 回合 | JAILED_WAIT_1 | 掷骰出狱（双骰局）/ 缴费 500 出狱 / 等待 |
| 第 2 回合 | JAILED_WAIT_2 | 掷骰出狱（双骰局）/ 缴费 500 出狱 / 等待 |
| 第 3 回合 | JAILED_WAIT_3 | **必须**缴费 500 金出狱（强制），若资金不足触发强制出售 |

> 关羽、司马懿技能：可在任意等待回合使用，免费出狱。

### 6.3 入狱期间其他规则

- **城池收入**：正常收取过路费（入狱玩家的城池仍有效）
- **卡牌**：入狱期间**不能主动使用**锦囊卡（步骤⑥跳过）
- **被动技能**：荀彧「筹谋」每回合+100金正常触发（被动不受影响）
- **手牌持有**：手牌保留，不强制弃牌
- **被卡牌攻击**：仍可被「火攻」「美人计」等卡牌影响

## 七、特殊裁定规则

### 7.1 效果叠加计算顺序

当多个倍率效果同时作用于过路费时，按以下顺序**乘法叠加**：

```
最终过路费 = 基础过路费 
           × 等级倍率 
           × 区域加成（荆州 1.2）
           × 进攻技能加成（张飞 1.5）
           × [减免处理] × 减免系数（刘备 0.5 / 夏侯惇 0.7）
           ÷ 劫营处理（甘宁：城主实收 × 0.5，甘宁获得 × 0.5）
```

**示例**：张飞在荆州的满级（5级）城池，刘备踩到：
- 基础：100 × 4（5级）× 1.2（荆州）× 1.5（张飞）= 720
- 刘备减免：720 × 0.5 = 360（刘备付 360）
- 张飞实收：360（不是 720，刘备技能先于城主技能）

> **设计意图**：减免技能针对付款方，加成技能针对收款方，两者独立计算后以付款方实际支付额为准，城主获得该额度（加成已体现在标价上）。

### 7.2 同时触发的处理顺序

当多个被动技能在同一时机触发时，按以下顺序处理：

1. 防御性/减免技能（空城计、刘备、夏侯惇）**先于**进攻性/加成技能（张飞）
2. 同类技能中，当前行动玩家的技能**先于**其他玩家的技能
3. 被动技能**先于**主动技能

### 7.3 资金为负的处理

游戏内不允许资金为负数：
- 任何扣款前先检查余额
- 不足时触发强制出售流程
- 强制出售后仍不足：触发破产（玩家剩余资金全部给城主，不足部分豁免）

### 7.4 「以逸待劳」与移动相关技能的互动

- 使用「以逸待劳」后：本回合不移动，获得 300 金，**跳过**步骤④⑤（不落地任何格子）
- 赵云重掷技能：本回合已使用「以逸待劳」则无法使用重掷（已跳过掷骰步骤）
- 声东击西（再掷一次）：已使用「以逸待劳」则无法使用（已跳过掷骰步骤）

## 八、常见规则裁定（FAQ 精确版）

**Q: 使用「瞒天过海」传送，传送到他人城池是否需要支付过路费？**  
A: 需要。传送落地后，格子效果正常触发。

**Q: 使用「走为上计」返回起点，是否获得起点奖励？**  
A: 获得（主动使用，视为经过起点）。

**Q: 「赤壁之战」让所有玩家随机移动，移动过程中经过起点是否给钱？**  
A: 不给钱。「赤壁之战」的强制移动为特殊位移，不触发途中格子效果（包括起点）。

**Q: 「连环计」的「A向B支付当前资金10%」，如果A资金为0怎么办？**  
A: 支付 0 金，卡牌效果照常（不触发破产）。

**Q: 多人同时破产（某张天命卡使多人同时无法存活）时如何排名？**  
A: 按资产从高到低排名（同时破产时，资产高者排名靠前）。资产相同则按城池数排，再相同则并列。

**Q: 诸葛亮预览锦囊卡后选择「不抽取」，下一个踩到锦囊格的玩家会抽到同一张吗？**  
A: 会。牌堆顶的牌未被移动，下一个踩格的玩家正常抽取。

**Q: 游戏中途玩家退出（非断线，主动退出），如何处理？**  
A: 等同于破产处理：城池释放为无主，手牌弃掉，从行动序列中移除。若只剩1人游戏，剩余1人直接获胜。
