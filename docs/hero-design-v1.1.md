# 三国大富翁 - 武将系统细化需求文档

**版本**：v1.1  
**更新日期**：2026-03-16  
**设计者**：虾仁  
**变更说明**：在 v1.0 基础上，补充所有武将技能的精确触发条件、触发时机、与其他效果的交互规则，以及前端实现所需的状态机细节

---

## 一、武将技能系统总览

### 1.1 技能类型分类

所有武将技能按**触发方式**分为三类，决定 UI 和服务端的处理逻辑：

| 类型 | 说明 | 处理方式 |
|------|------|----------|
| **全自动被动** | 满足条件时服务端自动结算，客户端只播放动画 | 无需玩家操作 |
| **提示型被动** | 满足条件时自动生效，但显示提示弹窗告知玩家 | 展示效果，无需选择 |
| **主动触发** | 满足条件时弹窗询问玩家是否使用，需玩家确认 | 玩家选择是/否 |
| **限次主动** | 同上，且有每局使用次数限制 | 同上，额外显示剩余次数 |

### 1.2 技能实现优先级（触发时机顺序）

当多个技能同时可以触发时，触发顺序：

```
1. 防御性技能（空城计、关羽出狱、司马懿出狱）
2. 全自动被动（刘备减免、夏侯惇减免、甘宁劫营）
3. 进攻性被动（张飞加成）
4. 信息型主动（诸葛亮预览）
5. 资源型被动（荀彧+100、曹操起点加成）
6. 回合结束技能（孙权换牌）
```

## 二、魏国武将技能精确规格

### 2.1 曹操 —「挟天子以令诸侯」

- **类型**：全自动被动
- **触发时机**：玩家棋子经过或停在起点格（格09）
- **精确效果**：在起点基础奖励之上，额外获得 15% 的起点基础奖励
  - 2-4人局：2000 × 15% = 300，共获得 2300 金
  - 5-6人局：2500 × 15% = 375，共获得 2875 金
  - 7-8人局：3000 × 15% = 450，共获得 3450 金
- **加速模式叠加**：加速模式的额外 +1000 金，曹操同样获得额外 15%（+150）
- **传送经过起点的处理**：
  - 「走为上计」传送到起点：**触发**（主动移动）
  - 「瞒天过海」传送到其他格（经过路径包含起点）：**不触发**（直接传送，无移动路径）
  - 「赤壁之战」随机移动经过起点：**不触发**（特殊位移）
- **前端展示**：经过起点时在曹操棋子旁弹出「+2300金」（而非普通玩家的「+2000金」），区分显示

---

### 2.2 司马懿 —「隐忍」

- **类型**：限次主动
- **每局次数**：1 次
- **触发时机**：玩家棋子因任何原因进入监狱格时，**在入狱前**触发询问弹窗
- **精确效果**：选择使用 → 不进入监狱，本次入狱尝试无效，回合继续正常执行
- **弹窗逻辑**：
  - 弹窗文本：「是否使用【隐忍】？可立即避免入狱（每局限1次，剩余次数：1）」
  - 确认使用：标记 `heroAbilityUsed.yinren = true`，不入狱，弹窗关闭
  - 取消 / 超时（10秒）：进入监狱，技能**不消耗**
- **与关羽的关系**：若玩家同时有司马懿技能（假设换皮），同一触发时机只出现一次弹窗（不存在两个技能同时触发的情形，因为一个玩家只有一个武将）
- **技能耗尽后**：`heroAbilityUsed.yinren = true`，弹窗不再出现，正常入狱

---

### 2.3 夏侯惇 —「拔矢啖睛」

- **类型**：全自动被动
- **触发时机**：玩家被要求支付过路费时（计算最终金额之后，扣款之前）
- **精确效果**：
  - 最终应付金额 = 计算结果 × 70%（即减少 30%）
  - 与刘备技能不叠加：若同时持有刘备效果（假设），取减免更大值（50%），不重复计算
  - 与甘宁劫营的交互：甘宁劫营在夏侯惇减免之后计算（见 §rules 效果顺序）
- **荆州区域加成的交互**：荆州区域过路费 +20%，夏侯惇 -30%，两者均生效（净效果为 0.84x）
- **前端展示**：费用面板显示「过路费：XXX → XXX（夏侯惇技能减免）」，划线原价，显示实付

---

### 2.4 张辽 —「突袭」

- **类型**：全自动被动
- **触发时机**：每次掷骰完成后，移动前
- **精确效果**：
  - 双骰规则：最终移动点数 = min(骰子总和 + 1, 12)
  - 单骰规则：最终移动点数 = min(骰子点数 + 1, 6)
- **与赵云重掷交互**：假设两人为不同玩家，互不影响；若同一玩家（游戏内不存在，预留说明）：重掷后再 +1
- **加速模式（骰子+2）的叠加**：加速模式 +2 与张辽 +1 叠加 = +3，但上限不变（双骰 ≤ 12，单骰 ≤ 6）
- **前端展示**：骰子结果显示原始点数，下方额外显示「张辽突袭：+1 → 实际移动 X 格」

---

### 2.5 许褚 —「虎卫」

- **类型**：全自动被动（游戏开始时一次性触发）
- **触发时机**：游戏开始，初始资金发放完成后
- **精确效果**：初始资金额外 +2000 金
  - 2人局：12000 + 2000 = 14000
  - 3-4人局：15000 + 2000 = 17000
  - 5-6人局：18000 + 2000 = 20000
  - 7-8人局：20000 + 2000 = 22000
- **与其他技能的交互**：仅初始加成，后续无效果，无交互问题
- **前端展示**：武将选择界面显示「初始资金 +2000」标注

---

### 2.6 荀彧 —「筹谋」

- **类型**：全自动被动
- **触发时机**：每个全局回合开始时（步骤①）
- **精确效果**：该玩家获得 100 金
- **入狱状态的交互**：入狱期间正常触发（入狱不影响被动收入）
- **乐不思蜀状态**：停留回合正常触发
- **美人计跳过回合**：被跳过的那个回合**不触发**（美人计跳过了回合开始步骤）
- **前端展示**：每回合开始时在荀彧棋子旁显示「+100金」小动画

## 三、蜀国武将技能精确规格

### 3.1 刘备 —「携民渡江」

- **类型**：全自动被动
- **触发时机**：刘备落地他人城池，过路费计算时
- **精确效果**：最终应付过路费 × 50%
- **与空城计的交互**：空城计优先级高于刘备技能。持有空城计时，优先弹窗询问是否使用空城计（完全免疫）；选择不使用或无空城计，则应用刘备 50% 减免
- **结盟期间**：盟友城池直接免过路费（高于刘备减免，无需触发刘备技能）
- **前端展示**：与夏侯惇同理，费用面板划线显示原价，标注「刘备仁德：50%减免」

---

### 3.2 诸葛亮 —「神机妙算」

- **类型**：主动触发
- **触发时机**：诸葛亮落地**锦囊格**时，在抽卡之前
- **精确效果**：
  1. 展示牌堆顶的下一张锦囊卡（只有诸葛亮玩家可见）
  2. 弹窗询问：「是否抽取？（当前显示：XXX卡）」
  3. 选择抽取 → 正常抽牌，卡牌加入手牌
  4. 选择跳过 → 牌堆顶的牌保持不动，诸葛亮本次不抽卡
- **牌堆为空时**：弃牌堆先重置洗牌，再展示牌顶
- **超时处理（15秒）**：自动选择「抽取」
- **其他玩家视角**：看到「诸葛亮正在使用神机妙算...」遮盖提示，不显示卡牌内容
- **前端展示**：诸葛亮专属的半透明卡牌预览弹窗（其他玩家看到模糊占位符）

---

### 3.3 关羽 —「过五关斩六将」

- **类型**：限次主动
- **每局次数**：1 次
- **触发时机**：同司马懿，在入狱前触发弹窗
- **精确效果**：选择使用 → 免费出狱，**且本回合可以继续正常行动**（掷骰移动）
  - 这是与司马懿的区别：司马懿是「不入狱」，关羽是「免费出狱」（效果相同，但描述不同以区分历史典故）
- **弹窗逻辑**：同司马懿，10秒超时不使用技能
- **已在监狱中的情形**：关羽技能只在「触发入狱瞬间」询问，不在已入狱后的回合询问
  - 已入狱玩家的出狱操作用「保释金」或「掷双数」，不用关羽技能

---

### 3.4 张飞 —「据水断桥」

- **类型**：全自动被动
- **触发时机**：他人玩家落地张飞所有城池时，过路费计算时
- **精确效果**：该城池对访客的过路费 × 1.5（即 +50%）
- **注意**：技能加成在等级倍率之后、区域加成之前（参见效果叠加公式）
- **与荆州区域加成叠加**：荆州城池 × 1.2（区域）× 1.5（张飞）= × 1.8 倍基础过路费
- **与刘备叠加**：刘备支付 = (基础 × 等级 × 1.5) × 0.5
- **前端展示**：他人踩到张飞城池时，费用数字颜色变红，标注「张飞据守：+50%」

---

### 3.5 赵云 —「七进七出」

- **类型**：限次主动（每回合 1 次）
- **每回合次数**：1 次（每个自己的回合重置）
- **触发时机**：赵云本人掷骰后，移动前
- **精确效果**：取消本次骰子结果，重新掷骰，使用第二次结果
- **弹窗逻辑**：
  - 掷骰动画结束后，弹窗显示「当前点数：X。是否使用【七进七出】重掷？」
  - 确认重掷：清除第一次结果，重新播放掷骰动画
  - 放弃 / 超时（10秒）：使用原始点数继续
- **重掷后结果不满意**：不可再次重掷（每回合限 1 次）
- **加速模式时**：加速模式+2 在重掷结果上叠加
- **前端展示**：掷骰结果显示后，若赵云玩家且本回合未用技能，高亮显示「重掷」按钮（10秒倒计时）

---

### 3.6 黄忠 —「百步穿杨」

- **类型**：全自动被动（带随机判定）
- **触发时机**：黄忠落地天命格，翻开天命卡并判定类型后，效果生效前
- **精确效果**：
  1. 翻开天命卡
  2. 若为「负面」类型：系统随机 30% 判定
     - 成功（30%）：将效果替换为对应正面效果（见 rules-v1.1.md §2.3 映射表）
     - 失败（70%）：效果照常生效
  3. 若为「正面」或「中性」类型：直接生效，黄忠技能不触发
- **前端展示**：
  - 触发替换时：天命卡先展示原负面卡图，然后动画翻转变为「黄忠神射！化解 XXX → 获得 XXX」
  - 未触发替换时：正常展示天命卡效果

## 四、吴国武将技能精确规格

### 4.1 孙权 —「制衡」

- **类型**：限次主动（每回合 1 次）
- **每回合次数**：1 次（每个自己的回合重置）
- **触发时机**：回合结束阶段（步骤⑧，手牌检查之前）
- **精确效果**：将手中 1 张锦囊卡放回弃牌堆（不进入牌堆顶），再从牌堆顶抽 1 张新卡
- **弹窗逻辑**：
  - 展示手牌，请玩家选择要换出的卡
  - 选定后展示新卡（已抽出，不可后悔）
  - 超时（15秒）：跳过使用
- **手牌为空时**：技能不触发（无牌可换）
- **换出的牌进入弃牌堆，不直接回到牌堆顶**（防止无限循环换同一张牌）
- **前端展示**：回合结束时，若孙权有手牌且本回合未使用，高亮「制衡换牌」按钮（15秒倒计时）

---

### 4.2 周瑜 —「火攻」

- **类型**：全自动被动
- **触发时机**：周瑜使用「火攻」锦囊卡时
- **精确效果**：「火攻」卡造成的伤害从 300 金变为 600 金
- **多张火攻卡**：每张都享受翻倍效果（2张火攻卡使用2次，每次600金）
- **锦囊卡数量**：「火攻」卡共 2 张，周瑜有可能集齐 2 张
- **与苦肉计的交互**：苦肉计不是火攻卡，周瑜技能不生效
- **前端展示**：使用火攻卡时，燃烧特效加强，数字显示「600金（周瑜：火攻翻倍）」

---

### 4.3 陆逊 —「火烧连营」

- **类型**：全自动被动
- **触发时机**：陆逊移动过程中，落地第 3 座（连续）他人城池
- **精确效果**：第 3 座城池免过路费
- **「连续」的定义**：陆逊在本次移动中，依次经过的城池（非连续停留，而是单次移动内的路径）
  - 例：骰子点数 6，路径上依次经过 A玩家城池→B玩家城池→C玩家城池，则 C 免费
  - 若路径中穿插了非城池格：连续中断，重新计数
- **「他人城池」的判定**：三座城池可以属于不同玩家，只要都不是陆逊自己的即可
- **边界**：若单次移动仅经过 1-2 座他人城池，技能不触发

> **实现说明**：服务端在处理陆逊移动时，需遍历移动路径上的每一格，按顺序统计他人城池数，每当计数达到 3 时，该城池免费，计数重置继续。

---

### 4.4 甘宁 —「百骑劫营」

- **类型**：全自动被动
- **触发时机**：甘宁落地他人城池，过路费计算时
- **精确效果**：
  - 甘宁实际支付：应付过路费 × 50%
  - 城主实际收入：应付过路费 × 50%（而非全额）
  - 即：甘宁「劫」走了 50%，城主少收 50%
- **与刘备技能的区别**：刘备是「减免」（少付，城主收全额）；甘宁是「劫营」（少付，城主也少收）
  - 实际上两者对甘宁的效果相同（都是付 50%），但城主收入不同
- **与空城计的交互**：空城计优先，完全免疫（甘宁不触发劫营）
- **前端展示**：支付过路费时，金币分为两份：一份飘向城主（50%），另一份留在甘宁处（50%），并显示「百骑劫营！」文字

---

### 4.5 太史慈 —「神射」

- **类型**：全自动被动
- **触发时机**：太史慈掷骰时
- **精确效果**：
  - 单骰规则（5-8人局）：骰子点数 +1（范围变为 2-7）
  - 双骰规则（2-4人局）：仅将最终总数 +1（范围变为 3-12）
- **与加速模式叠加**：加速模式+2 叠加在太史慈效果之上（+1+2=+3），但受上限约束
- **前端展示**：骰子显示原始点数，下方标注「太史慈神射：+1 → 实际 X 格」

---

### 4.6 鲁肃 —「慷慨」

- **类型**：双效（主动借贷 + 全自动被动保底）

**被动部分（保底收益）**：
- **触发时机**：每个鲁肃的回合开始（步骤①，与荀彧同时机）
- **精确效果**：鲁肃获得 50 金

**主动部分（借贷）**：
- **触发时机**：鲁肃的 CARD_ACTION 阶段（步骤⑥），或专属「借贷」操作按钮
- **操作流程**：
  1. 鲁肃点击「发起借贷」
  2. 选择目标玩家、借款金额（100-5000金，步长100）、利率（10% / 20% / 30%）
  3. 目标玩家收到借款请求，**可以拒绝**（10秒内响应）
  4. 同意：目标玩家立即获得借款金额，游戏记录还款计划
  5. 还款：X 回合后（借款时约定，1-5回合），自动从目标玩家资金扣除本金+利息，转给鲁肃
- **还款时目标玩家资金不足**：触发强制出售流程；若仍不足，鲁肃只收回剩余资金（坏账）
- **目标玩家破产**：债务清零，鲁肃损失本金
- **每局借贷上限**：鲁肃同时存在的借贷笔数 ≤ 3 笔
- **借贷期间鲁肃破产**：借贷合同取消，目标玩家不需还款
- **前端展示**：鲁肃的操作区额外显示「借贷管理」面板，显示当前各笔借贷状态

## 五、武将选择界面规格

### 5.1 选将界面展示信息

每个武将卡牌需展示：
- 武将立绘（半身像）
- 武将名称 + 称号
- 势力（颜色标识：魏=蓝、蜀=绿、吴=红）
- 稀有度（星级显示）
- 技能名称 + 简短描述（≤ 40 字）
- 锁定/解锁状态（未解锁时显示灰色遮罩 + 解锁条件）
- 已被其他玩家选择时：显示「已被 XX 选择」灰色遮罩

### 5.2 选将界面交互逻辑

```
进入选将界面
    │
    ├─ 展示所有 18 位武将，按已解锁/未解锁分区显示
    ├─ 默认高亮推荐武将（许褚/夏侯惇，新手友好）
    │
    ├─ 玩家点击武将 → 展示武将详情（技能完整说明 + 数值分析）
    │       ├─ 武将未解锁：显示「解锁后可选」，不可选择
    │       ├─ 武将已被他人选择：显示占用提示，不可选择
    │       └─ 可选择：显示「确认选择」按钮
    │
    ├─ 确认选择后：武将卡牌高亮，显示玩家名字标记，广播给其他玩家
    │
    └─ 60秒倒计时归零：未确认的玩家随机分配未被选择的默认解锁武将
```

### 5.3 武将解锁数据同步

解锁状态存储在用户账户级别，不随房间重置：

| 解锁条件 | 检测时机 |
|----------|----------|
| 默认解锁（9位） | 账户注册时初始化 |
| 首次游戏完成（3位） | 首局结算时检测 |
| 累计 5 局（3位） | 每局结算时检测 |
| 累计 10 局（3位） | 每局结算时检测 |
