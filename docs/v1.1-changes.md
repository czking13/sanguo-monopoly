## 一、规则变更（rules.md → rules-v1.1.md）

### 1.1 回合流程调整

| 模块 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 回合步骤 | 简单5步 | 精确8步（含入狱检查前置） | 状态机重写 |
| 入狱检查 | 回合开始后 | 步骤②独立检查 | 流程调整 |
| 美人计冷却 | 未定义 | 3全局回合冷却，以全局回合数计算 | 新增逻辑 |
| 手牌弃牌 | 超时未定义 | 30秒超时自动弃最旧卡 | 新增逻辑 |

### 1.2 骰子规则补充

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 双骰出狱 | 未明确 | 明确双数可出狱 | 新增逻辑 |
| 单骰出狱 | 未明确 | 1颗骰子无法通过掷骰出狱 | 新增逻辑 |
| 张辽上限 | 未明确 | 点数+1但不超过最大值（双骰≤12，单骰≤6） | 边界处理 |
| 太史慈单骰 | 未定义 | 范围变为2-7（效果+1，最低2） | 新增逻辑 |

### 1.3 锦囊卡使用规则

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 使用时机 | 未分类 | A类（主动阶段）/ B类（响应型） | UI 分类 |
| 每回合限制 | 1张 | A+B合计1张 | 逻辑调整 |
| B类多卡选择 | 未定义 | 弹出选择面板，超时跳过 | 新增UI |
| 响应卡时机 | 未明确 | 空城计/借刀杀人：过路费结算前；走为上计/瞒天过海：格子效果前 | 时序调整 |

### 1.4 过路费边界条件

| 场景 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 踩己方城池 | 不支付 | 不支付（不变） | - |
| 踩无主城池 | 不支付 | 不支付（不变） | - |
| 踩已破产玩家城池 | 未定义 | 视为无主城池 | 新增逻辑 |
| 城主入狱状态 | 未定义 | 仍收取过路费 | 新增逻辑 |
| 资金恰好等于过路费 | 未定义 | 支付后归零，不破产 | 边界处理 |
| 结盟期间踩盟友城池 | 未定义 | 免过路费 | 新增逻辑 |

### 1.5 城池规则调整

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 「大意失荆州」夺城等级 | 重置为1级 | **保留等级** | 逻辑调整 |
| 破产释放城池等级 | 未明确 | 重置为1级 | 新增逻辑 |
| 城池出售价格 | 未明确 | 累计投入×50%（含升级费） | 公式调整 |
| 升级时机 | 未明确 | 仅限步骤⑦ | 时序调整 |
| 连续升级 | 未定义 | 可以，每次单独确认 | 新增逻辑 |

### 1.6 联盟规则补充

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 触发方式 | 未定义 | 仅「桃园结义」天命卡 | 新增逻辑 |
| 目标拒绝权 | 未定义 | 无法拒绝（强制生效） | 新增逻辑 |
| 禁止卡牌 | 未定义 | 火攻/苦肉计/美人计/连环计/借刀杀人 | 新增逻辑 |
| 解除条件 | 未定义 | 3回合/使用禁止卡/破产/新结盟覆盖 | 新增逻辑 |

### 1.7 加速模式触发条件

| 条件 | v1.0 | v1.1 | 影响 |
|------|------|------|------|

### 1.8 监狱规则细化

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|

| 第3回合 | 未定义 | 强制缴费500出狱 | 新增逻辑 |
| 入狱期间技能 | 未定义 | 荀彧+100正常触发，不能用锦囊卡 | 新增逻辑 |
| 被攻击 | 未定义 | 仍可被火攻/美人计影响 | 新增逻辑 |

## 三、数值平衡变更（balance-design.md → balance-design-v1.1.md）

### 3.2 过路费计算公式

| 项目 | v1.0 | v1.1 | 影响 |
|------|------|------|------|

### 3.4 平衡测试基准

## 四、卡牌系统变更（card-design.md → card-design-v1.1.md）

### 4.1 牌堆管理

| 规则 | v1.0 | v1.1 | 影响 |
|------|------|------|------|
| 锦囊牌堆 | 未定义 | 16张，耗尽洗弃牌堆 | 新增逻辑 |
| 天命牌堆 | 未定义 | 20张，耗尽洗弃牌堆 | 新增逻辑 |
| 前端展示 | 未定义 | 显示剩余数量 | UI 新增 |

### 4.2 锦囊卡边界条件

| 卡牌 | 新增边界条件 |
|------|-------------|
| 火攻 | 扣除金额销毁（不转移），盟友不可选 |
| 连环计 | A资金不足付全部剩余，使用者不能作为A/B |
| 空城计 | 优先于甘宁劫营，10秒超时自动不使用 |
| 借刀杀人 | 替付方不足触发强制出售，城主只收实际额 |
| 苦肉计 | 使用前二次确认，使用者不足200无法使用 |
| 反间计 | 偷到的牌全局广播，手牌满时弃掉偷到的牌 |
| 美人计 | 3全局回合冷却，被跳过回合监狱等待仍计数 |
| 瞒天过海 | 传送到监狱不入狱，20秒超时不使用 |
| 围魏救赵 | 交换后入狱状态取消（可救盟友） |
| 走为上计 | 触发起点奖励，曹操额外15% |
| 声东击西 | 不撤销原格子效果 |
| 以逸待劳 | 不撤销已触发的格子效果，起点奖励保留 |

### 4.3 天命卡边界条件

| 卡牌 | 新增边界条件 |
|------|-------------|
| 赤壁之战 | 移动不给起点奖励，不触发落地效果 |
| 七擒孟获 | 无无主城池改为+500金，超时选编号最小 |
| 桃园结义 | 超时不触发（卡牌作废），新旧联盟立即切换 |
| 单刀赴会 | 资金不足只移动不购买，有主只移动付过路费 |
| 刮骨疗毒 | 资金<500扣光仍移除状态 |
| 大意失荆州 | 无城池改为-1000金，等级保留 |
| 火烧连营 | 与张飞叠加（×1.5×2=×3） |
| 乐不思蜀 | 步骤①②⑧正常执行，可被刮骨疗毒解除 |
| 望风而逃 | 后退不给起点奖励，落地监狱不入狱 |
| 威震华夏 | 仅当前回合有效 |
