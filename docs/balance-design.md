## 一、经济系统设计

### 1.1 资金流动模型

```
资金流入                    资金流出
┌─────────────┐            ┌─────────────┐
│ 经过起点    │ +2000      │ 购买城池    │ -600~2000  │
│ 出售城池    │ +50%购入价  │ 过路费      │ -60~200    │
│ 锦囊卡收益  │ +100~500   │ 升级城池    │ -300~2000  │
│ 天命卡收益  │ +200~500   │ 锦囊卡损失  │ -200~500   │
│ 武将技能    │ +100~2400  │ 天命卡损失  │ -100~500   │
└─────────────┘            │ 入狱费用    │ -500       │
                           └─────────────┘
```

### 1.2 初始资金设定

| 人数 | 初始资金 | 理由 |
|------|----------|------|
| 2人 | 12000 | 城池购买8折，资金需求降低 |
| 3-4人 | 15000 | 标准值 |
| 5-6人 | 18000 | 人多竞争激烈，需要更多资金 |
| 7-8人 | 20000 | 同上，加速模式需要更高资金池 |

### 1.3 城池价格体系

**设计原则**：
- 最低价格 = 初始资金的4%（600金）
- 最高价格 = 初始资金的13%（2000金）
- 过路费 = 购买价的10%

**价格分布**：

| 价格区间 | 数量 | 代表城池 |
|----------|------|----------|
| 600-800金 | 8座 | 濮阳、桂阳、绵竹、庐江等 |
| 900-1100金 | 12座 | 晋阳、新野、汉中、柴桑等 |
| 1200-1400金 | 8座 | 邺城、襄阳、永安、建业等 |
| 1500-1800金 | 3座 | 虎牢关、成都 |
| 2000金 | 1座 | 洛阳（起点） |

**总城池价值**：32800金

### 1.4 过路费计算

**基础公式**：
```
过路费 = 城池购买价 × 10% × 等级倍率 × 势力加成 × 技能加成
```

**示例**：
- 洛阳（2000金，1级）= 200金
- 洛阳（2000金，3级）= 400金
- 洛阳（2000金，5级）= 800金
- 洛阳 + 张飞技能 = 1200金

## 二、城池升级系统

### 2.1 升级费用与收益

| 等级 | 升级费用 | 过路费倍率 | 累计投入 | 回本需要（被踩次数） |
|------|----------|------------|----------|----------------------|
| 1级 | - | 1x | 购买价 | - |
| 2级 | 购买价×50% | 1.5x | 购买价×1.5 | 5次 |
| 3级 | 购买价×100% | 2x | 购买价×2.5 | 8次 |
| 4级 | 购买价×150% | 3x | 购买价×4 | 10次 |
| 5级 | 购买价×200% | 4x | 购买价×6 | 10次 |

**示例（洛阳2000金）**：

| 等级 | 升级费用 | 过路费 | 累计投入 |
|------|----------|--------|----------|
| 1级 | - | 200 | 2000 |
| 2级 | 1000 | 300 | 3000 |
| 3级 | 2000 | 400 | 5000 |
| 4级 | 3000 | 600 | 8000 |
| 5级 | 4000 | 800 | 12000 |

### 2.2 升级策略分析

**何时升级**：
- 城池位于高频路径 → 优先升级
- 城池位于低频路径 → 谨慎升级
- 资金充裕 → 可升级
- 资金紧张 → 优先保命

**升级优先级**：
1. 洛阳（起点，必经之地）
2. 关卡附近的城池
3. 锦囊/天命格附近的城池
4. 其他城池

| 借刀杀人 | 代付过路费 | ★★★ | 情境卡 |

| 反间计 | 偷取锦囊卡 | ★★★ | 情境卡 |
| 美人计 | 敌方停1回合 | ★★★★★ | 强控卡 |
| 瞒天过海 | 传送任意位置 | ★★★★ | 机动卡 |
| 围魏救赵 | 交换位置 | ★★★ | 策略卡 |

- 正面效果卡平均价值：300-400金

- 保命卡（空城计）数量少（2张）

## 四、不同人数平衡

### 4.1 2人局

**特点**：
- 对抗性强，无第三方干扰
- 游戏节奏可控
- 运气因素影响大

**平衡调整**：
| 项目 | 调整 | 理由 |
|------|------|------|
| 城池价格 | 8折 | 加快购买节奏 |
| 过路费 | +30% | 增加对抗强度 |
| 初始资金 | 12000 | 配合价格调整 |
| 骰子 | 2颗 | 加快移动 |
| 回合限制 | 30回合 | 防止僵局 |

**预期时长**：30-45分钟

**特点**：
- 标准体验
- 联盟与背叛
- 策略深度适中

**平衡调整**：
| 项目 | 调整 | 理由 |
|------|------|------|
| 城池价格 | 标准 | 标准值 |
| 过路费 | 标准 | 标准值 |
| 初始资金 | 15000 | 标准值 |
| 骰子 | 2颗 | 标准值 |

**预期时长**：45-60分钟

### 4.3 5-6人局

**特点**：
- 竞争激烈
- 城池供不应求
- 联盟重要

**平衡调整**：
| 项目 | 调整 | 理由 |

### 4.4 7-8人局

**平衡调整**：
| 项目 | 调整 | 理由 |

| 骰子 | 1颗 | 控制节奏 |

## 五、武将数值平衡

### 5.1 经济型武将收益对比

| 武将 | 技能收益 | 触发条件 | 稳定性 |
|------|----------|----------|--------|
| 曹操 | 2400金/圈 | 经过起点 | 极高 |
| 许褚 | 2000金（一次性） | 游戏开始 | 极高 |
| 荀彧 | 100金/回合 | 每回合 | 极高 |
| 鲁肃 | 200-600金/次 | 借贷成功 | 低 |

**40回合收益对比**：
- 曹操（10圈）：3000金（已调整为15%）
- 许褚：2000金
- 荀彧：4000金
- 鲁肃：2000-4000金（已增加保底）

**平衡结论**：
- 曹操已调整，保持现状
- 许褚保持不变
- 荀彧保持不变
- 鲁肃已增加保底收益

### 5.2 防守型武将收益对比

| 武将 | 技能收益 | 触发条件 | 稳定性 |
|------|----------|----------|--------|
| 刘备 | 50%过路费减免 | 踩敌方城池 | 高 |
| 夏侯惇 | 30%过路费减免 | 被收过路费 | 高 |
| 关羽 | 免费出狱1次 | 入狱 | 中 |

**假设踩到15次敌方城池，平均过路费100金**：
- 刘备节省：750金
- 夏侯惇节省：450金
- 关羽节省：约600金（避免3回合损失）

**平衡结论**：基本平衡，保持不变

### 5.3 机动型武将收益对比

| 武将 | 技能效果 | 价值评估 |
|------|----------|----------|
| 张辽 | 点数+1 | 约500金（更快到达目标） |
| 太史慈 | 点数2-7 | 约600金（更稳定） |
| 赵云 | 重掷1次 | 约800金（位置控制） |

**平衡结论**：
- 赵云略强，但策略性高，保持不变
- 张辽和太史慈各有优势，保持不变

### 5.4 进攻型武将收益对比

| 武将 | 技能收益 | 触发条件 | 稳定性 |
|------|----------|----------|--------|
| 张飞 | 50%过路费加成 | 城池被踩 | 中 |
| 甘宁 | 50%过路费返还 | 踩敌方城池 | 中 |
| 周瑜 | 火攻效果翻倍 | 使用火攻卡 | 低 |

**平衡结论**：基本平衡，保持不变

## 六、游戏节奏控制

### 6.1 标准局（4人）节奏

| 阶段 | 回合 | 主要活动 | 资金变化 |
|------|------|----------|----------|
| 开局 | 1-10 | 购买城池 | 15000→10000 |
| 发展 | 11-25 | 升级城池、收集卡牌 | 10000→12000 |
| 中期 | 26-40 | 激烈对抗 | 12000→8000（差距拉开） |
| 后期 | 41+ | 决胜负 | 一方破产 |

### 6.2 加速机制

**触发条件**：
- 7-8人局第20回合后
- 或任意局第50回合后

**加速效果**：
- 骰子点数+2
- 过路费+50%
- 起点奖励+1000金

### 6.3 防止僵局

**僵局判定**：
- 连续10回合无人破产
- 且所有玩家资金>10000金

**僵局处理**：
- 启动加速模式
- 或按总资产判定胜负
